# .argo — app-policies

## Purpose

Tiny public policy-site repository for app privacy policies and other legal
pages. This repo is separate from private app source repositories.

## Rules

- Keep only public legal/policy content here.
- Do not copy app source code, private assets, screenshots, credentials,
  release reports, or internal planning docs into this repo.
- Use one folder per app, e.g. `fartownik/privacy.html`.
- Publish with GitHub Pages from the repository root.

## Current Pages

- `fartownik/privacy.html` — Fartownik Privacy Policy.

## Expected Public URLs

If published as `sloik/app-policies` with GitHub Pages:

- `https://sloik.github.io/app-policies/`
- `https://sloik.github.io/app-policies/fartownik/privacy.html`

# App Policies

Public legal and policy pages for small apps.

This repository is intentionally separate from private app source repositories.
It is suitable for GitHub Pages hosting and should contain only public policy
text, not source code, private assets, credentials, screenshots, or internal
release artifacts.

## Pages

- Fartownik privacy policy: `fartownik/privacy.html`
- Fartownik support page: `fartownik/support.html`

## GitHub Pages

Recommended Pages URL after publishing this repository as `sloik/app-policies`:

```text
https://sloik.github.io/app-policies/
https://sloik.github.io/app-policies/fartownik/privacy.html
https://sloik.github.io/app-policies/fartownik/support.html
```

Enable GitHub Pages from the repository settings using the `main` branch and
repository root as the publishing source.
